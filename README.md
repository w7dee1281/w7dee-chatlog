#  Chat Log Wizard

GTA screen-style chat log generator.  
Create GTA-like chat logs with per-line coloring, font control , and export them as PNG images.

---

##  Live Demo
GitHub Pages:https://w7dee1281.github.io/w7dee-chatlog/


---

##  Features

###  Per-Line Color System
- Automatic color rules based on message type
- Manual override for **selected lines only**
- Manual colors always have **priority** over automatic rules

###  Manual Color Override
- Select one or multiple lines using the checkboxes next to the textarea
- Pick a color
- Press **Save**
- Only the selected lines change color

---

###  Font Control 
- Available fonts:
  - Tahoma (Default)
  - Roboto
  - Cairo
  - Tajawal
  - Poppins
  - Montserrat
---

###  Export Options
- Export as **PNG with background**
- Export as **transparent PNG**
- Colors, fonts, and sizes are preserved exactly
